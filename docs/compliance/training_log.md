# Training Log

**Document ID:** TRN-TRPM-LIMS-001
**Version:** 0.1 (DRAFT)

Every user who interacts with TRPM-LIMS in a regulated capacity must be
trained on their role-specific workflows and sign this log before being
granted production access.

## Training modules

| Module | Audience | Content |
|---|---|---|
| TRN-01 — Orientation | All users | Login, MFA setup, password policy, audit trail awareness |
| TRN-02 — Patient registration | Front desk | Patient registration, consent capture |
| TRN-03 — Accessioning | Lab techs | Sample receipt, barcode printing, workflow transitions |
| TRN-04 — Molecular workflows | Molecular techs | Plate layouts, NGS library prep tracking, instrument runs |
| TRN-05 — QC | QC team | QC metric recording, acceptance criteria |
| TRN-06 — Result review & e-signature | Directors / authorized signatories | Result review, electronic signature meaning (21 CFR 11.50) |
| TRN-07 — Reporting | Reporting team | PDF report generation, transmission to EMR |
| TRN-08 — System admin | IT / admins | User management, backups, incident response |
